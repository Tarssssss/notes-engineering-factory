# CLAUDE.md Update Protocol

## Daily (21:30 local time)

- Generate 1-3 lightweight optimization proposals.
- Do not edit `CLAUDE.md` directly.
- Proposal format:
  - 问题
  - 原因
  - 改进
  - 拟写入条目

## Weekly consolidation

- Merge accepted proposals.
- Remove duplicates and conflicts.
- Keep style rules concise and stable.
