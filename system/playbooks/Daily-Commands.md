# Daily Commands

Run from repository root:

```bash
node scripts/validate-notes.mjs
node scripts/sync-backlog.mjs
node scripts/weekly-focus.mjs
node scripts/review-check.mjs
node scripts/daily-claude-proposal.mjs
```

For AI suggestion draft:

```bash
node scripts/generate-ai-suggestion.mjs --note notes/books/deep-work-notes.md --todo "设计一周深度工作时间块并执行3天"
```
